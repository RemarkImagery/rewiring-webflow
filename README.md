# rewiring-webflow

Webflow Cloud + DevLink code components for Rewiring Aotearoa.

## Setup

1. `npm install`
2. `npm run dev` — local preview on localhost:3000
3. `npx webflow library share` — publish components to Webflow workspace

## Components

- **Arc Carousel** — Fan/arc card slider with Swiper.js, overlay, autoplay, CMS slot support
